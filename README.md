En este trabajo podemos encontrar una estructura con el siguiente contenido: 

- Una página index.html para la home de la web

&nbsp;

\- Una página PaginasInteres.html que contiene diferentes páginas web que suelen ser útiles para los informáticos en distintos campos de la informática ( Colaboración / Formación / Front-End ) 

&nbsp;

\- Tres páginas con puntos útiles para los informáticos. Con una página que contiene buenas prácticas que se recomiendan seguir, información sobre seguridad informática, y temas importantes de la informática que considero que todo informático debe conocer. 

&nbsp;

\- Una página con herramientas útiles para distintas funcionalidades como IDEs y editores de código, Herramientas de gestión y productividad, Extensiones de navegador útiles, y Herramientas para diseño y frontend.



\- Una página con información sobre mi biografía informática.

&nbsp;

\- Y por último una página de contacto que contiene 3 formularios distintos. 





En este proyecto he añadido información que considero que es importante para cualquier informático además de añadir puntos y herramientas útiles para nosotros. Es una página web que sigue en desarrollo por lo que iré añadiendo más información a medida que vaya trabajando en ella. 



Se han añadido enlaces a páginas externas, imágenes añadidas en una carpeta dentro de la estructura de la web, funcionalidades dentro de la web con JavaScript para darle dinamismo, y phps para el envío de los formularios de la web.
